# audax-selecao-api

    Desafio: Criação de uma API para realizar cadastros de artigos
      
      Requisitos:
          CRUD de usuários
          CRUD de artigos
          
      Tecnologias utilizadas:

          - Java 11
          - MariaDB Java Client
          - Spring Boot
          - Spring Web
          - Spring Security (UsernameNotFoundException)
          - Spring Boot DevTools
          - Validation
          - Slugify
